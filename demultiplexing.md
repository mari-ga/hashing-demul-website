| File      | Format         | Type |
|:-------------|:------------------|:------|
| HTO matrix | 10x folder | filtered matrix |
| HTO matrix | h5         | raw matrix |
| HTO matrix | 10x folder | raw matrix|
| RNA matrix | 10x folder | filtered matrix|
